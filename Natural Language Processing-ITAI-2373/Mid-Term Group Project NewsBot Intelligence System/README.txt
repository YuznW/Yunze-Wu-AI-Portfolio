Midterm — NewsBot Intelligence System (Group Project)
Project Title

NewsBot Intelligence System — Midterm Version

Problem Statement

To build a prototype NLP system capable of classifying or extracting information from news text using classical NLP tools.

Approach & Methodology

Performed text preprocessing (tokenization, lemmatization, stop-word removal).

Applied TF–IDF or bag-of-words for feature extraction.

Trained a classifier or rule-based system for news interpretation.

Results & Evaluation

Prototype successfully processed sample news inputs.

Demonstrated reasonable classification accuracy.

Learning Outcomes

Reinforced understanding of traditional NLP pipelines.

Experience in team-based AI system design.
